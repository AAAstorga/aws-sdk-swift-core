# aws-sdk-swift-core

A Core Framework for [AWSSDKSwift](https://github.com/noppoMan/aws-sdk-swift)

## Package.swift

```swift
import PackageDescription

let package = Package(
    name: "MyAWSApp",
    dependencies: [
        .Package(url: "https://github.com/noppoMan/aws-sdk-swift-core", majorVersion: 0, minor: 2)
    ]
)
```
